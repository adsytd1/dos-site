Задача

Forte Bank — один из крупнейших банков Казахстана. Руководители подразделений ставили задачи сотрудникам через разрозненные каналы — email, мессенджеры, устно. Задачи терялись, сроки срывались, контроль выполнения занимал часы рабочего времени HR-менеджеров.

Нужна была единая система: постановка задач → контроль сроков → автоматические уведомления → отчётность.

Решение

Мы создали Telegram-бота для внутренних коммуникаций Forte Bank. Система позволяет руководителям ставить задачи сотрудникам прямо в Telegram, отслеживать статус выполнения и получать автоматические отчёты.

  • Постановка задач через Telegram — руководитель описывает задачу, назначает исполнителя и дедлайн
  • Автоматические уведомления исполнителям с деталями задачи
  • Система напоминаний — за 24 часа, за 2 часа до дедлайна
  • Отчёты о статусе задач — ежедневные и еженедельные сводки для руководства
  • Эскалация просроченных задач — автоматическое уведомление вышестоящему руководству
  • Интеграция с внутренними системами банка

Техническая архитектура

  • Telegram Bot API — основной интерфейс для руководителей и сотрудников
  • n8n — автоматизация workflow (уведомления, напоминания, эскалация)
  • PostgreSQL — хранение задач, статусов, истории изменений
  • Система ролей — разграничение доступа (руководитель / исполнитель / наблюдатель)
  • Автоматическая отчётность — формирование сводок по подразделениям
  • Безопасность — шифрование, доступ только для авторизованных сотрудников
